INFORMATION:

Tabs separate mutually exclusive content.
Only one tab is active at a time and therefore only the content related to the active tab is visible at any given time.

Dependency:

This plugin is only dependent on jQuery.

Usage:

Use the following HTML Markup:
<nav role='navigation'>
  <ul>
    <li>
      <a href="#tab-id1" class="active">Tab1</a>
    </li>
    <li>
      <a href="#tab-id2">Tab2</a>
    </li>
  </ul>
</nav>
<div id="tab-id1" class="active">
  <h3>This is tab 1.</h3>
</div>
<div id="tab-id2">
  <h3>This is tab 2.</h3>
</div>

And in your html file, call the plugin as:
$('nav').responsiveTab();
PS: To select a tab by default, add class "active" to the particular tab and it's content div.
